如何用 hugo 搭建个人博客

June 24, 2020

如何用 hugo 搭建个人博客(Windows版)

一、安装 hugo

  1. 下载 Hugo

在 Hugo 的 Github 下载 release 版本

  1. 解压，将 hugo.exe 放到一个目录下，并将该目录添加到 Path
  2. 重启终端，运行 hugo version 查看版本

示例

二、快速搭建博客

  1. 进入 Hugo 官网，点击Quick Start快速开始
  2. 从 Step 2 开始抄代码，直到 Step 7
Step 2: Create a New Site
    hugo new site 项目名称 (建议取成github账号名称.github.io-creator)

Step 3: Add a Theme
    cd 项目名称
    git init
    git submodule add https://github.com/budparr/gohugo-theme-ananke.git themes/ananke
    echo 'theme = "ananke"' >> config.toml

Step 4: Add Some Content
    hugo new posts/博客名称.md

Step 5: Start the Hugo server
    hugo server -D

Step 6: Customize the Theme
    baseURL = "github上的博客地址"
    languageCode = "zh-Hans"
    title = "博客名称"
    theme = "ananke"

Step 7: Build static pages
    hugo -D

三、将本地博客配置到 Github

  1. cd 项目名称（用hugo命令行创建的项目）
  2. 新建 .gitignore 文件，将 public 文件忽略
  3. cd public
  4. git init 初始化 git 仓库
  5. 在 Github 新建仓库，名称为 账号名称.github.io
  6. 将本地仓库和远程仓库进行关联
  7. 将本地仓库的内容上传到 Github
  8. 在 Github 仓库的 setting 的最下方，点击这个地址就能访问 Github Pages 的博客了!

示例

四、如何创建第二篇博客

  1. 在 xxx.github.io-creator 目录（注意确保自己不在 public 目录）里运行 hugo new posts/博客名称.md
  2. 运行 code posts/第二篇博客.md 对文件进行编辑，注意不要把文件原本的内容给删了，直接在后面另起一行写新内容。
  3. 运行 hugo -D，得到新的 public 目录
  4. 进入 public 目录，执行一下操作
    • git add . 注意有一个点
    • git commit -m update
    • git push -f (其中 -f 是强制上传的意思)
  5. 等待几分钟后，你的博客就会出现第二篇文章了！

五、如何在 Hugo 里插入图片

方法一

  1. 打开 sm.ms 网站，将图片上传上去，得到图片的 url
  2. 在文章里使用 语法即可显示图片

方法二

  1. 将图片文件 1.png 存放到 static 目录里，得到 static/1.png
  2. 在博客 Markdown 里写 就能看到 static/1.png 对应的图片了
  3. 如果你想在 static 里创建 images 目录，直接创建就好，对应的图片前面加上 /images 即
    • 将图片文件 2.png 放到 static/images 目录里，得到 static/images/2.png
    • 在博客里写 ![](/images/2.png) 即可
